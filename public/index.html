This static frontend is served by Netlify. Backend interactions are proxied to an external Python/Gradio deployment via redirects.

Setup required: Replace YOUR_BACKEND_URL in public/_redirects and netlify.toml before deploying.

Why this approach?

  • Netlify does not natively run persistent Python servers like Gradio.
  • We host the interactive Python app elsewhere (Render, HuggingFace Spaces, Railway, Fly.io).
  • Netlify provides fast global CDN delivery for the static assets.

Next Steps

  1. Deploy your Python app.py (Gradio) to an appropriate host.
  2. Update redirect target to that URL.
  3. Commit & push; connect repo to Netlify; deploy.
